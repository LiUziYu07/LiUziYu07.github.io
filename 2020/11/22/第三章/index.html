运算方法

定点数的加减法运算以及溢出判断

####补码 加减法运算

加法：

[X+Y]补=[X]补+[Y]补[X+Y]_{补} = [X]_补 + [Y]_补[X+Y]补​=[X]补​+[Y]补​

减法：

[X−Y]补=[X]补+[−Y]补[X-Y]_补 = [X]_补 + [-Y]_补[X−Y]补​=[X]补​+[−Y]补​

求[Y]补[Y]_补[Y]补​技巧：直接对[Y]补[Y]_补[Y]补​连同符号位在内取反，然后在最低位加1

基本器件：全加器FA

补码运算结果的溢出判断

  • 方法1

    检测操作数和运算结果的符号位：

    V=X0Y0S0‾+X0‾Y0‾S0V = X_0Y_0\overline{S_0}+\overline{X_0}\overline{Y_0}S_0V=X0​Y0​S0​​+X0​​Y0​​S0​

    VVV得1时表示运算结果溢出

  • 方法2

    运算时最高有效位的进位为C1C_1C1​，符号位的进位为C0C_0C0​，则溢出检测电路为：

    V=C0⊗C1V=C_0\otimes C_1V=C0​⊗C1​

  • 方法3

    采用变形补码：

    正数符号位为00，负数符号位为11

    [X]补=Xf1Xf2.X1...Xn[X]_补 =X_{f1}X_{f2}.X_1...X_n[X]补​=Xf1​Xf2​.X1​...Xn​

    上溢出：双符号位为01

    下溢出：双符号位为10

    溢出判断：V=Xf1⊗Xf2V = X_{f1}\otimes X_{f2}V=Xf1​⊗Xf2​

移码的加减法运算

加法：

[X+Y]补=[X]移+[Y]补[X+Y]_补 = [X]_移 + [Y]_补[X+Y]补​=[X]移​+[Y]补​

减法：

[X−Y]补=[X]移+[−Y]补[X-Y]_补 = [X]_移 + [-Y]_补[X−Y]补​=[X]移​+[−Y]补​

####移码的溢出判断

采用双符号位判断

移码的双符号位：第一个符号位为0，第二个符号位正数为0，负数为1

当运算结果的第一个符号位为0时，无溢出

运算结果的第一个符号位为1时，溢出

定点数乘法运算

手工乘法

正常计算即可

####原码一位乘法运算方法

部分积+右移

####补码的一位乘法运算方法

在运算开始时，Yn+1Y_{n+1}Yn+1​补成0

算法：

  • 如果Yn+1=YnY_{n+1} = Y_{n}Yn+1​=Yn​，部分积加0，并右移一位结果
  • 如果Yn+1<YnY_{n+1} < Y_nYn+1​<Yn​，部分积加[−X]补[-X]_补[−X]补​，并右移一位结果
  • 如果Yn+1>YnY_{n+1} > Y_nYn+1​>Yn​，部分积加[X]补[X]_补[X]补​，并右移一位结果

阵列乘法器：

大量与门和FA构成的并行阵列乘法器

###浮点数运算

####加减法

基本步骤：

  • 0操作数检查

    如果有一个操作数为0，则只需要对符号位进行检查

  • 对阶

    移动阶码小的浮点数的尾数，使两个数的阶码相同

    对齐的步骤

    • 求阶差ΔE\Delta EΔE
    • 将阶码小的那个浮点数的尾数向右移动ΔE\Delta EΔE,并同时将其阶码加1

  • 尾数运算

    用补码加减法法则，计算尾数

  • 结果的规格化

    • 左规格化：尾数结果为00.0Φ...Φ00.0\Phi...\Phi00.0Φ...Φ和11.1Φ...Φ11.1\Phi...\Phi11.1Φ...Φ,尾数向左移动1位且阶码减1，直到尾数形式为：00.1Φ...Φ00.1\Phi...\Phi00.1Φ...Φ或11.0Φ...Φ11.0\Phi...\Phi11.0Φ...Φ
    • 右规格化：尾数的结果为01.Φ...Φ01.\Phi...\Phi01.Φ...Φ或10.Φ...Φ10.\Phi...\Phi10.Φ...Φ，尾数向右移动且阶码加1，直到尾数形式为:00.1Φ...Φ00.1\Phi...\Phi00.1Φ...Φ或11.0Φ...Φ11.0\Phi...\Phi11.0Φ...Φ

  • 舍入

    0舍1入：如果向右移出的是1，则在最低位加1；否则，不处理

    恒置1:只要数字位1被移掉，就将最后一位恒置为1

  • 溢出处理

    阶码上溢：阶码的符号位为01

    阶码下溢：阶码的符号位为10

####乘除法

阶码的运算采用补码/移码的加法法则

尾数的运算采用补码的乘法法则

  • 循环移位

##运算器

暂存器

用于存放参与计算的数据及运算结果，只被控制器硬件逻辑控制或微程序所访问

通用寄存器组

用于存放程序中用到的数据，可以被软件设计者所访问

状态寄存器组

累加器

数据通路

内部总线：用于连接各个部件的信息通道，传输过程中的数据

###运算器分类

  • 并行运算器

    能对一个数据运算器所有位进行处理的运算器

  • 串行运算器

    一次只处理一位

  • 串/并行运算器

    把一个完整的数据分成若干段进行计算
